**Goal:**
Given a list of words, repeatedly remove words that are *anagrams* of their previous adjacent word until no such pairs remain.

---

### âœ… Code

```python
class Solution:
    def removeAnagrams(self, words: List[str]) -> List[str]:
        res = [words[0]]
        for i in range(1, len(words)):
            if sorted(words[i - 1]) == sorted(words[i]):
                continue
            res.append(words[i])
        return res
```

---

### ğŸ§  Explanation

* **`sorted(word)`** â†’ rearranges the letters alphabetically (e.g., `"baba"` â†’ `['a','a','b','b']`).
* Two consecutive words are **anagrams** if their sorted forms are equal.
* If they are anagrams, we **skip** adding the current word.
* Otherwise, we **append** it to the result list.
* The first word is always included since thereâ€™s nothing before it.

### ğŸ§¾ Example

Steps:

2. `"baba"` is an anagram of `"abba"` â†’ skip
3. `"bbaa"` is an anagram of `"baba"` â†’ skip
4. `"cd"` is not an anagram â†’ add `"cd"`
5. Next `"cd"` is same as previous â†’ skip

Output:

```python

```
